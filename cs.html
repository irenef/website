Computer Science

Here are some of my cs-related work in no particular order.

-Beethoven-

An interactive processing project that allows user to click and drag to rotate the 3D model of a Beethoven bust about an arbitrary axis (3D model source: Princeton 3D Model Search Engine).

-Face Detection-

Uses Histogram of Oriented Gradients techniques, non-maximum suppression, linear regression and logistic regression to train a model that detects human faces.

-Undergraduate Senior Honors Thesis-

Guided by Dr. Matthew Hibbs, this research aimed to develop an algorithm that estimates the camera position in space from which an image was created using computer vision techniques. 3 major steps: defining a distinct combination of features of the object, recognizing the object with the distinct features, and calculating camera position using the mapping information between the projected 2-D image and the 3-D object.

-Visualization of Gene Expression-

Visualized gene expression data of two species, danio rerio and mus musculus, under normoxia and hypoxia conditions using single-species scatter plots and two-species proximity maps. The visualization results highlight the similar responses to oxygen level in terms of gene expression across species.
